# WonderBurger

2018TokyoGameShowで展示した学生ゲーム作品です。
これはグループ制作作品で、主にプログラミング、3Dモデルと3Dモーションを担当しました。
制作期間：2018年4月から7月程度
制作メンバー数：3人

ジャンル：シューティング仕分けゲーム
主にゲームパッドを使用して遊ぶ用に制作しました。

【PCでの操作方法】
基本マウス操作と方向キーのみ
・PCゲームの方向キー(ASDW)で、カーソル移動&カメラ視点移動
・左クリックでハンバーガー発射
・右クリックでポテト発射
・ホイールクリックでドリンク発射
・スペースで、スキル発動(青い円ゲージMax時)
